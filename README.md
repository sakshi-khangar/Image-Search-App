# Image-Search-App
A simple Image Search web app built using HTML, CSS, and JavaScript. It uses the Unsplash API to fetch and display images based on user search queries, with a clean UI and a “Show More” feature for pagination.

## Features
- Search images by keyword
- Images fetched from Unsplash API
- Show More button to load additional images
- Clean and responsive UI
- Handles empty search and no-result cases

## Technologies Used
- HTML5 – Structure of the app
- CSS3 – Styling and layout
- JavaScript (ES6) – Logic & API handling
- Unsplash API – Image data source

## Unsplash API Setup
- Create an account on Unsplash Developer
- Generate an Access Key
- Replace the access key in Image_Search.js
const accessKey = "YOUR_UNSPLASH_ACCESS_KEY";

## How to Run the Project
- Download or clone the repository
- Open index.html in your browser
- Enter a keyword (example: nature, cars, dogs)
- Click Search
- Click Show More to load more images
